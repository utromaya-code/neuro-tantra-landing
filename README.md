# Лендинг «Нейрофизиология тантры»

Одностраничный лендинг курса-ретрита «Нейрофизиология тантры» по техническому заданию. Готов к публикации на **GitHub Pages**.

---

## Как быстро получить ссылку (без кода)

**Способ 1 — Netlify Drop (≈1 минута)**  
1. Откройте в браузере: **https://app.netlify.com/drop**  
2. Перетащите папку `neuro-tantra-landing` (или только файлы `index.html` и `README.md` из неё) в окно браузера.  
3. Netlify выдаст ссылку вида `https://random-name-12345.netlify.app` — это и есть ваш лендинг.

**Способ 2 — GitHub Pages**  
См. раздел «Публикация на GitHub Pages» ниже. После пуша ссылка будет:  
`https://ВАШ_ЛОГИН.github.io/neuro-tantra-landing/`

## Содержание лендинга

1. **Hero** — заголовок, УТП, CTA, инфо-строка (8 занятий / 15 сессий / 2 недели)
2. **Для кого** — 5 карточек с аудиторией
3. **Тантра без эзотерики** — текст + таблица «тантрический термин ↔ наука»
4. **Программа** — аккордеон (8 теоретических тем) + блок про 15 практик
5. **Результаты** — 6 карточек
6. **Подходит / не подходит** — две колонки (зелёная / красная)
7. **Об авторе** — Леонид Кутузов, фото, цитата, достижения
8. **Формат и условия** — 3 колонки (теория, практика, сопровождение) + стоимость
9. **FAQ** — аккордеон с 8 вопросами
10. **CTA + форма** — призыв записаться, форма (имя, email, телефон, чекбокс)
11. **Футер** — контакты, реквизиты ИП, ссылки на документы

## Публикация на GitHub Pages

### Вариант 1: репозиторий только для лендинга

1. Создайте новый репозиторий на GitHub (например, `neuro-tantra-landing` или `username.github.io` для персональной страницы).

2. Клонируйте репозиторий и скопируйте в него файлы:
   ```bash
   git clone https://github.com/YOUR_USERNAME/neuro-tantra-landing.git
   cd neuro-tantra-landing
   # Скопируйте сюда index.html (и при необходимости папку assets, если добавите)
   ```

3. Включите GitHub Pages:
   - **Settings** → **Pages**
   - **Source**: Deploy from a branch
   - **Branch**: `main` (или `master`), папка **/ (root)**
   - Save

4. Закоммитьте и запушьте:
   ```bash
   git add index.html README.md
   git commit -m "Landing: Нейрофизиология тантры"
   git push origin main
   ```

5. Сайт будет доступен по адресу:  
   `https://YOUR_USERNAME.github.io/neuro-tantra-landing/`  
   (или `https://YOUR_USERNAME.github.io/` если репозиторий называется `username.github.io`).

### Вариант 2: лендинг в подпапке существующего репозитория

Если у вас уже есть репозиторий (например, с другими проектами):

- Создайте папку, например `neuro-tantra`, и положите в неё `index.html`.
- В **Settings** → **Pages** выберите ветку и папку, где лежит этот каталог (или корень).
- Тогда лендинг будет по адресу:  
  `https://YOUR_USERNAME.github.io/REPO_NAME/neuro-tantra/`

## Что нужно настроить перед запуском

1. **Форма регистрации**  
   В `index.html` найдите форму с `action="https://formspree.io/f/YOUR_FORM_ID"`.  
   - Зарегистрируйтесь на [Formspree](https://formspree.io/), создайте форму и подставьте свой **Form ID** вместо `YOUR_FORM_ID`.  
   - Либо замените `action` на URL вашей CRM (AmoCRM, Битрикс24 и т.д.) и при необходимости измените метод/поля.

2. **Цена и даты**  
   Замените плейсхолдеры в тексте:
   - `[укажите цену]` — актуальная стоимость
   - `[дата]` — старт курса и дедлайн ранней регистрации
   - `[X] мест` — количество мест при необходимости

3. **Политика возврата**  
   В блоке FAQ в ответе на вопрос «Можно ли получить возврат средств?» вставьте ваш текст вместо `[Укажите вашу политику возврата]`.

4. **Фото автора**  
   Сейчас используется заглушка с Unsplash. Замените `src` у изображения в блоке «Об авторе» на путь к реальному фото (можно положить файл в папку `assets` и указать `assets/author.jpg`).

5. **Аналитика** (по желанию)  
   Добавьте перед `</head>` код счётчиков Яндекс.Метрики и Google Analytics (GA4) по инструкциям ТЗ.

6. **Документы**  
   Создайте страницы или PDF: Политика конфиденциальности, Договор оферты, Согласие на обработку персональных данных — и проставьте ссылки в футере.

## Локальный просмотр

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
# Python 3
python3 -m http.server 8000

# или npx
npx serve .
```

Сайт будет доступен по адресу: `http://localhost:8000`
